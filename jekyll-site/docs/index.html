Quick start

Your first PDF in two minutes

Majorsilence.Pdf is a tiny, self-contained PDF library for .NET. Add one package, create a document, draw some text and tables, and save. No native binaries, no extra dependencies.

Installation

A single package — no companions required:

# the whole library, nothing else to add
dotnet add package Majorsilence.Pdf

Hello, PDF

Create a PdfDocument, add a page, draw, and save. The document is IDisposable — wrap it in using.

using Majorsilence.Pdf;

using var doc = new PdfDocument();
var page = doc.AddPage(PageSize.A4);

page.Text("Hello, PDF", x: 72, y: 72, size: 24);
page.Line(72, 96, 520, 96);

doc.Save("hello.pdf");

Text, tables & images

The drawing surface is fluent. Set a font once and chain calls, or pass a table of rows and let the layout flow across pages.

var page = doc.AddPage(PageSize.Letter);

page.SetFont(Font.Helvetica, 11)
    .Text("Invoice #1024", 72, 72, size: 20)
    .Image(logoBytes, x: 430, y: 64, width: 110);

page.Table(rows, 72, 130, columns: new[] { "Item", "Qty", "Total" });

Save or stream

Write to a path, or to any Stream — ideal for returning a PDF straight from an ASP.NET endpoint.

// to disk
doc.Save("report.pdf");

// to a stream — e.g. an HTTP response
await doc.SaveAsync(Response.Body);
