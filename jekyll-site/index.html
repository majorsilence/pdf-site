PDF for .NET.
Tiny by design.

A modern, self-contained PDF library for .NET 8 & 10. Zero dependencies, no native binaries — one small managed package. Tri licensed MIT OR Apache-2.0 OR BSD-3-Clause, pick your choice.

Get started Read the docs
$ dotnet add package Majorsilence.Pdf
Program.csin 
using var doc = new PdfDocument();
var page = doc.AddPage(PageSize.A4);

page.Text("Sales Summary — Q2 2026", 72, 72, size: 22);
page.Table(rows, 72, 120);

doc.Save("report.pdf");
